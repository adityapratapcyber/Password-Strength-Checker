# Password Strength Checker

This is a simple Python project that checks whether a password is weak, medium, or strong.

I created this project while learning Python and basic cybersecurity concepts.

## How it works

The program checks:

- Password length
- Uppercase letters
- Numbers

Based on these checks, it classifies the password as:

- Weak Password
- Medium Password
- Strong Password

## Sample Input

```
Password123
```

## Sample Output

```
Strong Password
```
